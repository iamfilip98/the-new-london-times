  • Select Cells: Click any cell to select it

  • Game Recovery: Resume unfinished puzzles when you return

How does the competition work?
Each day, both players compete by completing the same three Sudoku puzzles (Easy, Medium, Hard). The player with the highest total score wins the day. Scores are calculated using linear time scaling, harsh error penalties (15% per error), and strategic hint penalties (0.5%-3%). Winners of individual difficulty levels get a 30% bonus. Win streaks, achievements, and overall records are tracked automatically.

The Amazing Puzzle Algorithm

How are the daily puzzles created?
Our advanced puzzle generation system creates unique, high-quality Sudoku puzzles using sophisticated algorithms:
Puzzle Generation Process
  1. Deterministic Seeding: Uses the date to create a unique seed, ensuring both players get identical puzzles
  2. Complete Solution Generation: Creates a valid 9√ó9 Sudoku solution using backtracking with seeded randomization
  3. Strategic Clue Removal: Intelligently removes numbers while maintaining puzzle solvability and uniqueness
  4. Difficulty Calibration: Adjusts clue count ranges (Easy: 35-42, Medium: 19-25, Hard: 18-25 clues)
  5. Advanced Validation: Ensures puzzles require appropriate solving techniques for their difficulty level
  6. Quality Assurance: Multiple validation passes guarantee puzzle integrity and unique solutions

What makes our algorithm special:

  • Deterministic Generation: Same seed = same puzzle for fair competition
  • Unique Solutions: Every puzzle has exactly one valid solution
  • Logical Solvability: Medium/Hard puzzles require advanced techniques like naked pairs, hidden subsets, etc.
  • Consistent Difficulty: Rigorous calibration ensures appropriate challenge levels
  • Database Storage: Puzzles are stored in PostgreSQL for consistency and performance
How is puzzle difficulty determined?
  • Easy (35-42 clues): Solvable with basic techniques (naked singles, hidden singles)
  • Medium (19-25 clues): Requires intermediate techniques (naked pairs, hidden pairs, basic patterns)
  • Hard (18-25 clues): Demands advanced techniques (naked triples, hidden triples, X-Wing patterns, complex deductions)
Our algorithm validates that each puzzle can be solved using only the techniques appropriate for its difficulty level.

Security & Privacy

Is my account secure?
Absolutely! We implement industry-standard security measures:
  • Password Encryption: All passwords are hashed using bcrypt (2^10 rounds = 1,024 hashing iterations)
  • No Plaintext Storage: Passwords are never stored in readable form
  • Database-Backed Auth: Secure PostgreSQL authentication system
  • API-Based Login: All authentication happens server-side with secure validation
  • Session Security: Authentication expires when you close the browser for your protection
  • Environment Variables: Sensitive configuration stored securely, never in code
How can I change my password?
Passwords are managed securely by the system administrator through environment variables. If you need to change your password, contact the system administrator who can update it using the secure initialization system. This ensures maximum security and prevents unauthorized password changes.
What data do you store?
We store only essential game data:
  • User Accounts: Username, encrypted password hash, display name
  • Game Progress: Puzzle completions, times, scores, errors, hints used
  • Achievements: Unlocked achievements and timestamps
  • Statistics: Win/loss records, streaks, performance metrics
  • Game States: Auto-saved puzzle progress for resume functionality

Privacy Note: No personal information, email addresses, or tracking cookies are collected. Your data is used solely for game functionality and competition tracking.
